  5. Modules

    6. Utilisation dans l'algorithme :

    algo :nomAlgo Ord Lexique
    Module(paramètres) nomModule(module)

    Forme du module
    module :nomModule(p1:tp1;...;pn:tpn) Ord Lexique
    instructions déclarations (sauf paramètres)

    : entrée (non modifiable)
    : sortie
    : entrée / sortie
    p : paramètre
    tp : type du paramètre

  7. Tableaux
    1. Une dimension

      2. Déclaration (lexique) :
      T_nomTypeTableau(type) = table[borneInf..borneSup]:typeTableau
      nomTableau(T_nomTypeTableau)
      Accès aux données :
      nomTableau[placeTableau]

    2. Plusieurs dimensions

      3. Déclaration (lexique) :
      T_nomTypeTableau(type) = table[borneInf1..borneSup1, ..., borneInfn..borneSupn]:typeTableau
      nomTableau(T_nomTypeTableau)
      Accès aux données :
      nomTableau[placeTableau1, ..., placeTableaun]

  8. Type scalaire / intervalle

    9. Ensemble de val ordonné, il y a deux fonctions prédéfinies : Pred()etSucc()

    1. Scalaire

      2. Déclaration (lexique) :
      T_nomType(type) = (val1, ..., valn)
      exemple : T_Jour(type) = (lundi, mardi, mercredi, jeudi, vendredi, samedi, dimanche)
      nomVar(T_nomType)

    2. Intervalle

      3. Déclaration (lexique) :
      T_nomType(type) = borneInf .. borneSup
      nomVar(T_nomType)

  9. Type structure

    10. Déclaration (lexique) :
    nomTypeStructure(type) = structure(nomChamp1(nomType1); ...; nomChampn(nomTypen))
    nomVarStructure(nomTypeStructure)
    Chaque nomChamp peut être plusieurs variables (exemple : "nom, prenom(chaine)")
    Accès à une donnée :
    nomVarStructure.nomChamp[i]
