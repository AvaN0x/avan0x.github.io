Projet CPOA

CPOA : Conception et Programmation Objet Avancées

Le projet est disponible sur github.com/AvaN0x/ricatte_sublet_cpoa.

Présentation

Le but du projet était de créer une application de gestion de commandes pour des "pulls moches de noël".

Travail effectué

Travail effectué en binôme avec github.com/oxypomme

Contenu technique

  • Java
  • JAVAFX
  • Interface Graphique
  • Application console
  • POO : Programmation Orienté Objet
  • Design Pattern DAO : Data Access Object
  • MVC : Modele View Controller
  • JDBC : Java Database Connectivity
  • Tests unitaires
  • Sauvegarde Mémoire
  • Sauvegarde SQL

Avis

C'est un très bon projet pour découvrir la puissance des DAO.
